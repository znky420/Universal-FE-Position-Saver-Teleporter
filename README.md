# zink Save & Teleport Script

Ein funktionales Teleport-Tool für Roblox, entwickelt von zink. 

## Features

* **Key System**: Gesichert durch ein einfaches Key-System (Key: `meow`).
* **Positionsspeicherung**: Speichert Koordinaten unter benutzerdefinierten Namen.
* **Teleport-System**: Schnelle Reise zu allen gespeicherten Wegpunkten.
* **Verwaltung**: Löschen einzelner Einträge oder vollständiger Reset der Liste.
* **Utilities**: Enthält einen WalkSpeed-Slider.
